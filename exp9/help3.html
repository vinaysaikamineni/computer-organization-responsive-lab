Single Cycle MIPS CPU

Create Your Own Instruction

This simulator provides a unique feature of allowing the user to create his own instruction. The hardware is fixed but by changing the control signals new instructions can be made. Create your own setup asks the user to choose from a r-type or i-type and then set the 7 control signals according to his need. In this version the user can create a maximum of 2 new instructions (one of each type).


Eg instruction -

myinst r31, r5, 0


with control signals -

0, 1, 0, 1, 3, 1, 0


NOTE- All the "dont-care" signals have been assumed to be "0".
